Frequently used git commands

git clone [repository, e.g. github url]

git status
git diff

git commit -am "commit message"
  (make sure there are no untracked files with git status or otherwise this will add them to the repo )
git push
git pull

git branch [branchname]
git checkout [branchname]
git fetch origin [originbranch]:[localbranch] (I use the same name)

git stash (remove changes but saves them)
git stash pop (pulls stashed changes [after checking out another branch e.g.])
git merge [use sparingly and with caution]
git cherry-pick [commit] (merges a single commit)
